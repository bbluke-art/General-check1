Thai Language Learning App

Helping foreigners master Thai tones and phrases

Project Overview

A mobile-first application designed to make learning Thai accessible and fun.

Features include tone recognition, phrase practice, and cultural context for better understanding.

Technologies Used

React Native Node.js MongoDB TensorFlow.js

Features

Tone Trainer

Interactive exercises to master the five Thai tones with voice recognition.

Daily Phrases

Essential phrases for travelers and expats with audio pronunciation.

Culture Notes

Contextual learning with cultural insights and etiquette tips.
